# - Terminal Portfolio

Un portfolio web interactivo con estilo terminal moderno, diseñado para GitHub Pages.

## 🚀 Características

- **Interfaz de Terminal**: Navegación mediante comandos de texto
- **Diseño Responsivo**: Optimizado para todos los dispositivos
- **Animaciones Suaves**: Efectos visuales modernos y sutiles
- **Sin Dependencias**: HTML5, CSS3, JavaScript vanilla

## 📁 Estructura del Proyecto

```
Portfolio/
├── index.html          # Página principal
├── styles.css          # Estilos CSS con tema terminal
├── script.js           # Lógica JavaScript interactiva
└── README.md           # Este archivo
```

## 🎮 Comandos Disponibles

Navega por el portfolio usando estos comandos en la terminal o los botones de navegación superiores:

| Comando | Descripción |
|---------|-------------|
| `help` | Muestra todos los comandos disponibles |
| `whoami` | Información sobre Josué Moreno |
| `ls projects/` | Lista de proyectos |
| `skills --list` | Muestra habilidades técnicas |
| `contact --help` | Información de contacto |
| `clear` | Limpia la terminal |
| `exit` | Cierra la terminal |
| `restart` | Reinicia la terminal |

### Comandos Alternativos

También puedes usar estos alias:
- `ls` en lugar de `ls projects/`
- `skills` en lugar de `skills --list`
- `contact` en lugar de `contact --help`
- `about` o `resume` en lugar de `whoami`
- `projects` en lugar de `ls projects/`

### Easter Eggs...

Prueba estos comandos secretos:
- `sudo` - Mensaje de permiso denegado divertido
- `matrix` - Referencia a The Matrix
- `hello world` - Saludo clásico
- `date` - Muestra fecha y hora actual
- `pwd` - Directorio actual
- `ls -la` - Listado de archivos detallado


## 🔧 Troubleshooting

### Problemas Comunes

**Q: Los comandos no funcionan**
A: Asegúrate de que JavaScript esté habilitado en tu navegador

**Q: El diseño se ve extraño en móvil**
A: Recarga la página y verifica que tengas conexión a internet

**Q: GitHub Pages no muestra el sitio**
A: Verifica que el archivo `index.html` esté en la raíz del repositorio

**Q: Los colores no se ven bien**
A: Algunos navegadores bloquean fuentes externas; la fuente Fira Code se carga desde Google Fonts
